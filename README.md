Moscow Real Estate Analysis & Price Prediction (2020–2026)
Project Overview

This project focuses on Exploratory Data Analysis (EDA) and Machine Learning on Moscow real estate market data from 2020 to 2026. The objective is to analyze housing market trends, understand factors affecting property prices, and build a regression model for price prediction.

The project includes data cleaning, visualization, feature analysis, and regression modeling using Python libraries.

Dataset Information

The dataset contains real estate market details such as:

District names
Property prices
Rental prices
Mortgage rates
Listing counts
Monthly price changes
New-build and secondary housing prices
Objectives
Perform data cleaning and preprocessing
Analyze real estate price trends
Compare district-wise property prices
Visualize housing market patterns
Identify important features affecting prices
Build a regression model for price prediction
Evaluate model performance using R² Score
